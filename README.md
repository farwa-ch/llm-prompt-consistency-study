# LLM Prompt Variations Study

## Summary
In this project, I will be testing the effect of varying the phrasing in a prompt on the output that is generated from the large language model (LLM).

## Project Goals
To determine the effect of prompt variations on the following factors:
- response consistency 
- factual accuracy 
- hallucinations

## Process
- Constructed a test data set consisting of factual and reasoning-based prompts
- Created variations of each prompt 
- Measured consistency through semantic similarity 
- Examined patterns of variance based on prompt type 

## Findings
- Factual prompts had high consistency between outputs 
- Reasoning prompts varied significantly 
- Adversarial prompts decreased accuracy
- Structured prompts increased consistency
